Malone data sets from:

https://sites.google.com/site/bmmalone/files/urlearning

This folder contains the data files necessary to reproduce
the experiments in Fan et al., AAAI 2014 and Fan et al., UAI
2014. In particular, it contains the (preprocessed) csv files
(csv) and the MDL scores in a flat text format usable by the
c++ version of URLearning and GOBNILP (pss).
